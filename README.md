# Nextly - Landing Page Template with Next.js & Tailwind CSS

United Work It is a Software Engineering Company. 
Landing Page Its built with Next.js & TailwindCSS.
And its completely open-source.
 
![image](https://uwork.rf.gd/company.png)

## How to use

Click on the [Use Template](https://github.com/UEWork/UWorkCompany) button on this page to clone this respository to your github account. Or you can also clone this respository using terminal or bash. 

### 1\. Clone this Repository

```bash
git clone https://github.com/UEWork/UWorkCompany.git
```

### 2\. Navigate to the directory

```
cd united-work
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
